## 香滑卤鸡腿
### 第一次做
参考的是小米电饭煲的“香滑卤鸡腿”食谱
* 原材料

材料|数量|备注
---|---|---
冻鸡腿|860g|5个
生姜|3片
香叶|3片
八角|3个
料酒|适量|焯水用
红烧汁|37g
蒜|3瓣
干辣椒|8颗
花椒|少量

* 做法
    1. 鸡腿加水，加热，慢慢解冻，加料酒焯水
    2. 把材料都放入电饭煲，加水到5刻度线，加热60分钟

* 总结
加热到30分钟的时候，取出来，是比较滑，但是味道比较淡，颜色是淡黄色  
第二次炖60分钟后，里面的水基本都干了，味道不错了，但是酱浓，肉还是有一点淡
